# 🍅 Minimalist Pomodoro Timer

A simple and clean Pomodoro timer built using HTML, CSS (Tailwind), and JavaScript.

## 🚀 Features
- ⏱ 25-minute work timer
- 🧘 5-minute break timer (selectable)
- ▶️ Start / ⏸ Pause / 🔁 Reset controls
- 🔔 Alarm sound when timer ends
- 🎨 Minimal UI with smooth animations
- 🌌 Aesthetic background with glassmorphism effect
- ⚡ Accurate timing using real-time tracking (works even when tab is inactive)

## 🛠 Tech Stack
- HTML
- Tailwind CSS
- JavaScript

## 📦 How to Run
1. Clone the repository
2. Open `index.html` in your browser
